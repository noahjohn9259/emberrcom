WP-Admin Icons provides WordPress admins the ability to customize WordPress admin panel icons through an easy-to-use interface.

== Description ==

If you're tired of the standard WordPress icons and want to give your WordPress admin interface a icon-makeover then look no further.  This plugin is packed with icons for you to choose from or if you have your own you can use that too!  Once you install the plugin you will have the ability to change all your WordPress admin icons through an easy-to-use and intuitive GUI interface.

This plugin was created to give WordPress admins and easy tool to swap out the WordPress Admin Panel icons.  If you're developing a site for a client and want to provide them that extra customization then this plugin will work great.  Use your own or choose from one of 2,000+ icons included with this plugin.


= Features =

1. Easy-to-use Interface.
2. 2,000+ pre-packaged admin icons.
3. Works with WP 3.0's custom post types.
4. Works out of the box - no setup needed.
5. No need to modify WordPress files.
6. Doesn't require other plugins to work.
7. Uploader built-in to use your own icons.
8. Choose to hide or show your admin icons.
 

== Installation ==

1. Upload `wp-admin-icons` folder and it's contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You should now be able to access the Plugin's options via your settings panel.

Note: If you have WordPress 2.7 or above you can simply go to 'Plugins' &gt; 'Add New' in the WordPress admin and search for "WP Admin Icons" and install it from there.

== Frequently Asked Questions ==

= Why should I use this plugin? =

Use this plugin if you want to customize your WordPress admin interface's icons to reflect your site's needs.  Suppose you have a lot of custom post types and you want to custom the icons to make the interface more intuitive.  This plugin will do just that and more.
